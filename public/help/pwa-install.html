Install as Local App (PWA)

Meshbex CAD Explorer supports installation as a Progressive Web App (PWA). Once installed, the application is cached locally in your browser and can be launched and used without an internet connection.

Requirements

  • A Chromium-based browser – Google Chrome, Microsoft Edge, or Brave. Firefox does not support PWA installation.

  • An internet connection for the first installation. After that, no internet is needed.

How to Install

  1. Open your browser and navigate to https://meshbex.com.

  2. Wait for the page to fully load.

  3. Look for the install icon in the browser address bar on the right side – it looks like a computer with a download arrow (⊕).

  4. Click it and confirm by clicking Install.

  5. A shortcut will appear on your desktop and/or in the Start menu.

Alternatively, use the browser menu:

  • Chrome – menu (⋮) → Save and share → Install as app…

  • Edge – menu (⋯) → Apps → Install this site as an app

Launching After Installation

Double-click the desktop shortcut. The application opens in its own standalone window – without the browser address bar – and works fully offline.

How Offline Mode Works

On the first visit, the browser downloads and caches the entire application (JavaScript, CSS, icons, demo models, help files). All subsequent launches load from this local cache, so no internet connection is required.

When you are online, the cache is silently updated in the background so you always get the latest version.

Sharing the App with Another Computer

The PWA cache cannot be copied between computers. To install on another machine:

  1. Connect that computer to the internet (even briefly).

  2. Open https://meshbex.com in Chrome or Edge.

  3. Click the install icon and install.

  4. The computer can then be used offline.

Uninstalling

  • Open the app, click the menu (⋮ / ⋯) → Uninstall.

Notes

  • Imported models (loaded via Import GLB) are kept in memory for the current session only – they are not cached permanently.

  • Clearing browser data or cache will remove the installed PWA cache. You will need to be online for the next launch to re-cache.

  • The PWA remains installed indefinitely as long as it is used occasionally. Browsers may evict unused PWA caches after several weeks of inactivity.
